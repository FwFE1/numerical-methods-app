ИНСТРУКЦИЯ ПОЛЬЗОВАТЕЛЯ

Программа: Численные методы: Маев — Вариант 8

1. Назначение программы

Программа предназначена для выполнения расчётной работы по дисциплине «Численные методы».

В приложении реализованы численные методы по варианту 8. Пользователь может выбирать задания, изменять исходные данные, выполнять расчёты и просматривать результаты в виде чисел, таблиц и графиков.

2. Запуск программы

Для запуска готовой версии приложения необходимо открыть файл:

NumericalMethodsApp.exe

После запуска появится окно загрузки. Затем откроется главное окно программы.

3. Главное окно программы

В левой части окна находится меню со списком заданий.

Для открытия нужного задания необходимо нажать на соответствующую кнопку в боковом меню.

В центральной части окна отображаются:
- название задания;
- условие задачи;
- исходные данные;
- элементы для изменения параметров;
- кнопки расчёта и сброса;
- результаты вычислений;
- таблицы;
- графики, если они предусмотрены заданием.

4. Выбор задания

В программе доступны следующие задания:

1. Метод Гаусса и метод простой итерации.
2. Метод прогонки и метод Зейделя.
3. Метод вращения для нахождения собственных значений и собственных векторов.
4. Уточнение корня уравнения методами простой итерации, половинного деления и Ньютона.
5. Интерполяционные многочлены Лагранжа и Ньютона.
6. Кубические сплайны дефекта 1.
7. Аппроксимация методом наименьших квадратов.
8. Нахождение производных по таблице.
9. Метод Симпсона.
10. Решение задачи Коши методами Рунге-Кутта 4-го порядка и Адамса.
11. Решение краевой задачи методом прогонки.

5. Изменение исходных данных

Исходные данные можно изменять вручную.

В зависимости от задания пользователь может:
- вводить числа в поля ввода;
- изменять значения в таблицах;
- задавать точность вычислений;
- изменять шаг расчёта;
- изменять параметры уравнений или систем.

После изменения исходных данных необходимо нажать кнопку расчёта, чтобы получить обновлённый результат.

6. Выполнение расчёта

Для выполнения расчёта необходимо:

1. Выбрать задание в боковом меню.
2. Проверить или изменить исходные данные.
3. Нажать кнопку "Рассчитать".
4. Просмотреть полученные результаты.

Результаты могут отображаться в виде:
- итоговых численных значений;
- таблиц итераций;
- промежуточных коэффициентов;
- графиков.

7. Сброс исходных данных

В заданиях может быть предусмотрена кнопка "Сбросить" или "Сбросить таблицу".

Она возвращает исходные данные к значениям по умолчанию для варианта 8.

8. Работа с таблицами

Некоторые задания используют таблицы для ввода и вывода данных.

Если таблица доступна для редактирования, пользователь может изменить значения прямо в ячейках таблицы.

После редактирования таблицы необходимо повторно нажать кнопку расчёта.

9. Работа с графиками

В заданиях, где требуется визуализация, программа строит график автоматически после выполнения расчёта.

График отображается внутри окна выбранного задания.

10. Возможные ошибки

Если введены некорректные данные, программа может выдать ошибку или не выполнить расчёт.

Рекомендуется проверять:
- чтобы числовые поля не были пустыми;
- чтобы в таблицах были введены числа;
- чтобы шаг и точность были положительными;
- чтобы размеры матриц соответствовали условию задачи.

11. Завершение работы

Для завершения работы достаточно закрыть главное окно программы.

12. Запуск из исходного кода

При необходимости программу можно запустить из исходного кода.

Для этого необходимо установить зависимости:

pip install PySide6 numpy pandas matplotlib

Затем выполнить команду:

python main.py

13. Сборка программы

Для сборки приложения в EXE используется PyInstaller.

Команда сборки:

pyinstaller --clean --onefile --windowed --name NumericalMethodsApp --icon "assets/app_icon.ico" --add-data "assets;assets" main.py

Готовый файл будет создан в папке dist.
