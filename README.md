# Численные методы: Маев — Вариант 8

## Описание проекта

Данное приложение разработано для выполнения расчётной работы по дисциплине «Численные методы».

Программа представляет собой desktop-приложение с графическим интерфейсом, в котором реализованы основные численные методы по варианту 8. Все задания доступны из одного окна через боковое меню.

Приложение позволяет:
- выбирать нужное задание через меню;
- просматривать условие задачи;
- изменять исходные данные;
- выполнять перерасчёт;
- получать численные результаты;
- просматривать таблицы вычислений;
- строить графики для заданий, где это предусмотрено.

## Реализованные задания

1. Решение системы линейных алгебраических уравнений методом Гаусса и методом простой итерации.
2. Решение системы линейных алгебраических уравнений методом прогонки и методом Зейделя.
3. Нахождение собственных значений и собственных векторов методом вращения.
4. Уточнение корня нелинейного уравнения методами простой итерации, половинного деления и Ньютона.
5. Интерполяция функции многочленами Лагранжа и Ньютона.
6. Построение кубических сплайнов дефекта 1.
7. Аппроксимация функции методом наименьших квадратов.
8. Нахождение производных по таблично заданной функции.
9. Численное интегрирование методом Симпсона.
10. Решение задачи Коши методами Рунге-Кутта 4-го порядка и Адамса.
11. Решение краевой задачи методом прогонки.

## Возможности программы

- Работа с 11 отдельными заданиями в одном приложении.
- Редактирование исходных данных.
- Автоматический перерасчёт результатов.
- Вывод промежуточных и итоговых таблиц.
- Построение графиков.
- Единый тёмный визуальный стиль интерфейса.
- Возможность запуска как из исходного кода, так и в виде собранного `.exe` файла.

## Используемые технологии

Приложение разработано на языке Python.

Используемые библиотеки:
- PySide6 — графический интерфейс;
- NumPy — численные вычисления;
- Pandas — работа с таблицами;
- Matplotlib — построение графиков;
- PyInstaller — сборка приложения в `.exe`.

## Запуск из исходного кода

Для запуска проекта из исходного кода необходимо установить зависимости:

```bash
pip install PySide6 numpy pandas matplotlib
```

После установки зависимостей запустить файл:

```bash
python main.py
```

## Сборка в EXE

Для сборки приложения в исполняемый файл использовался PyInstaller.

Команда сборки:

```bash
pyinstaller --clean --onefile --windowed --name NumericalMethodsApp --icon "assets/app_icon.ico" --add-data "assets;assets" main.py
```

Готовый файл создаётся в папке:

```text
dist/
```

## Запуск готового приложения

Для запуска готовой версии необходимо открыть файл:

```text
NumericalMethodsApp.exe
```

После запуска появляется окно загрузки, затем открывается главное окно приложения.
